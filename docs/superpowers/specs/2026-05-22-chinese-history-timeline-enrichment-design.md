# 中华朝代长卷史料扩充与学术深度重塑设计规范 (Chinese History Dynasty Timeline Enrichment Design)

## 1. 目标与背景 (Goal & Background)
针对现有中华历史朝代大纲 `src/data/history-timeline.json` 进行全方位的学术史料扩容与深度润色。
目前，每个大一统朝代的时间节点仅 4~6 个，且描述流于通俗。本项目致力于依据《史记》、《资治通鉴》、钱穆《国史大纲》、吕思勉《中国通史》及《剑桥中国史》等一流学术典籍，对数据进行全面升级：
1. **扩展朝代主旨描述 (Dynasty Descriptions)**：将通俗介绍重写为蕴含深刻制度变迁、社会经济和文化思潮演进的学术散文。
2. **倍增大事记里程碑 (Milestones)**：为每个大一统或主流朝代精选 **8~12 个核心里程碑事件**，覆盖政治、军事、经济、文化思想、科技、外交等多个维度。
3. **完全兼容架构 (Backward Compatibility)**：保留现有 JSON 数据架构和字段命名规则，确保前端渲染及搜索过滤零报错。

---

## 2. 史料来源与学术视域 (Historical References & Intellectual Angles)
在编写和润色过程中，严格汲取并贯彻以下学术视角：
- **政治制度**：汲取钱穆《中国历代政治得失》中关于汉代察举制、唐代三省六部与科举、明废宰相设内阁、清设军机处等皇权与相权制衡的理性剖析。
- **社会经济**：借鉴白寿彝《中国通史》及黄仁宇《中国大历史》关于土地税制改革（如两税法、一条鞭法、摊丁入亩）的实证分析。
- **思想文化**：结合《史记》、《资治通鉴》，展现百家争鸣、罢黜百家独尊儒术、玄学与佛道交融、理学心学合流、近现代启蒙运动的思想激荡。
- **地缘与民族**：借鉴哈佛《中国帝制史》和《剑桥中国史》，秉持客观唯物的历史观，中正叙述中原王朝同北方游牧民族（匈奴、契丹、女真、党项、蒙古、满洲等）的地缘博弈与和平融合，生动书写中华民族“多元一体”的发展脉络。

## 3. 数据契约与约束 (Data Schema & Constraints)
为保障前端 `src/js/main.js` 能毫无阻碍地渲染，扩展后的 JSON 必须完美符合以下字段约束：

### 3.1 字段规范 (Fields Specification)
- `id` (string, unique)：朝代唯一标识。
- `name` (string)：中文名称。
- `ename` (string)：英文名称。
- `type` (string)：标识类型，`dynasty`（大一统/主王朝）或 `concurrency`（并立分裂期）。
- `startYear` (number)：开始公元年份（公元前使用负数）。
- `endYear` (number)：结束公元年份。
- `capital` (string, optional)：都城，若为 `concurrency` 类型则在 `states` 数组中描述。
- `founder` (string, optional)：开国创立者，若为 `concurrency` 类型则在 `states` 数组中描述。
- `color` (string)：十六进制主题色，维系现有国风配色体系。
- `description` (string)：深度学术历史叙述。
- `milestones` (array of objects, optional, only for `type: "dynasty"`)：大事记里程碑列表：
  - `year` (string)：年份文本表示（如“前202年”、“105年”）。
  - `title` (string)：大事件标题。
  - `desc` (string)：事件发生背景、过程及深远历史意义。
  - `tags` (array of strings)：标签类别，可选范围：`["政治", "军事", "文化", "思想", "科技", "外交", "经济", "制度", "社会"]`。
- `states` (array of objects, optional, only for `type: "concurrency"`)：并立割据期政权明细：
  - `name` (string)：政权名（如“曹魏”）。
  - `capital` (string)：都城。
  - `founder` (string)：开国君主。
  - `desc` (string)：政权特点与地缘政治经济概述。
  - `color` (string)：政权专属识别色。

## 4. 各历史时期史料扩充详案 (Detailed Timeline Roadmap)

### 4.1 先秦时期 (`pre-qin`, `type: "dynasty"`)
*   **主述深化**：叙述从“部落联盟”向夏商周三代“宗法封建制”的演进，剖析青铜礼乐的建构，及春秋战国铁器牛耕催生的社会剧变。
*   **里程碑 (8个)**：
    1. 前2070年 夏朝建立（家天下确立）
    2. 前1600年 鸣条之战（商汤建商）
    3. 前1046年 牧野之战（武王克商，西周确立）
    4. 前1042年 周公制礼作乐（宗法制与分封制的核心体系化）
    5. 前770年 平王东迁（周室衰微，春秋大戏开启）
    6. 前551年 孔子诞生与百家争鸣（先秦思想的黄金时期）
    7. 前356年 商鞅变法（秦国富国强兵的绝对转折）
    8. 前256年 都江堰修筑（关中与天府之国的地缘农业革命）

### 4.2 秦朝 (`qin`, `type: "dynasty"`)
*   **主述深化**：聚焦专制集权体制的创设、郡县制的确立、思想与度量衡的一统，以及由于急政暴役导致二世而亡的历史警示（《过秦论》）。
*   **里程碑 (7个)**：
    1. 前221年 天下一统（首个皇帝制与郡县制帝国确立）
    2. 前214年 蒙恬北击匈奴（收复河套与万里长城修筑）
    3. 前213年 焚书坑儒（思想一统的极端实践）
    4. 前210年 沙丘之变（胡亥夺位与秦政崩塌序曲）
    5. 前209年 大泽乡起义（平民起义的壮丽第一声）
    6. 前207年 巨鹿之战（项羽摧毁秦廷主力）
    7. 前207年 子婴投降与秦朝覆灭（短暂大一统的落幕）

### 4.3 两汉 (`han`, `type: "dynasty"`)
*   **主述深化**：阐释“汉承秦制”并在汉武帝时期实现“罢黜百家独尊儒术”的思想一统，剖析汉代贤能察举参政制，以及东汉光武中兴下的豪强地主世族演进。
*   **里程碑 (10个)**：
    1. 前202年 西汉建立（刘邦即位，与民休息）
    2. 前154年 七国之乱（平定叛乱，武帝集权的前置铺垫）
    3. 前138年 张骞出使西域（丝绸之路开拓）
    4. 前134年 独尊儒术（太学设立与儒学正统地位确立）
    5. 前119年 漠北之战（卫青霍去病重创匈奴，漠南无王庭）
    6. 前91年 司马迁修《史记》（首部纪传体通史巨作）
    7. 9年 新朝建立与王莽改制（托古改制的悲剧性尝试）
    8. 25年 光武中兴（东汉建立，重塑汉室正统）
    9. 105年 蔡伦改进造纸术（文明载体走向世界廉价普及）
    10. 184年 黄巾起义（世族豪强兼并下的社会崩塌与三国割据大幕）

### 4.4 三国时期 (`three-kingdoms`, `type: "concurrency"`)
*   **主述与 states 润色**：深度刻画曹魏的“九品中正制”与“屯田制”、蜀汉的“隆中对”与诸葛亮治蜀的法制实践、东吴对江南工商业与海外（夷洲）的航海开发，揭示分裂中蕴含的融合力量。

### 4.5 晋朝与十六国 (`jin-dynasty`, `type: "concurrency"`)
*   **主述与 states 润色**：剖析西晋统治阶层奢侈与“八王之乱”引发的五胡内迁，东晋衣冠南渡下的“门阀政治”（王与马共天下）和南方经济开发，以及北方十六国的剧烈民族融合和前秦淝水之战的战略逆转。

### 4.6 南北朝时期 (`southern-northern`, `type: "concurrency"`)
*   **主述与 states 润色**：重墨刻画南朝（宋齐梁陈）的庶族兴起与佛教文化激荡，北方北魏拓跋珪一统中原后“孝文帝汉化改革”对北方少数民族全面融入中华文明圈的决定性作用，阐明其为隋唐大一统所铺就的关陇集团制度基础。

### 4.7 隋朝 (`sui`, `type: "dynasty"`)
*   **主述深化**：概述隋文帝结束四百年分裂、创立三省六部制度及输籍法之功；辩证阐述隋炀帝大运河之宏图以及三征高句丽过度压榨民力的骤亡悲剧。
*   **里程碑 (6个)**：
    1. 581年 隋朝开国（杨坚禅周，确立开皇之治底座）
    2. 589年 南朝灭陈（终结近四百年分裂，天下一统）
    3. 605年 创立科举（进士科设，人才向上流动的伟大变革）
    4. 605年 京杭大运河开凿（跨南北经济地理命脉畅通）
    5. 612年 三征高句丽惨败（倾巢远征，社会矛盾集中引爆）
    6. 618年 隋朝灭亡（杨广江都遇刺，隋亡唐兴）

### 4.8 唐朝 (`tang`, `type: "dynasty"`)
*   **主述深化**：展现贞观、开元盛世的宏博包容，丝路商贸与万国来朝。剖析均田制与府兵制的互表关系，以及安史之乱后藩镇割据与杨炎两税法对帝国中晚期社会结构的重组。
*   **里程碑 (10个)**：
    1. 618年 唐朝开国（李渊建唐，收束乱世）
    2. 626年 玄武门之变与贞观之治（李世民纳谏听风，开创守成治世）
    3. 641年 文成公主入藏（中藏民族和亲，文明的交流）
    4. 690年 武周代唐（女皇武则天执掌乾坤，开创贞观遗风）
    5. 713年 开元盛世（唐朝国力、人口与文学史诗的黄金盛景）
    6. 755年 安史之乱爆发（盛唐急速滑坡，藩镇割据，社会南移）
    7. 780年 杨炎推行两税法（均田废，按资产征税，税制划时代转型）
    8. 819年 韩愈与古文运动（重振儒家道统，吹响宋代理学的前哨）
    9. 878年 黄巢起义（世族社会秩序的大洗牌）

*   **主述与 states 润色**：深度叙述两宋“重文轻武”治国方略的利弊（内部政治文明极盛、思想理学大成、工商业繁盛与科技三大发明的全球溢出，但对外军事积弱），以及与契丹辽、女真金、党项西夏并立共存的多中心国际格局。

## 5. 质量审查与防范机制 (Quality Control & Verification Rules)
1. **彻底查禁占位符**：文件中严禁出现任何诸如 `TBD`、`TODO`、`等`、`此处待补充` 类的非生产性占位内容。所有文字和年份均为绝对真实、学术详实的定稿文字。
2. **严防 JSON 语法错误**：扩充后的大型 JSON 文件必须经由工具（如 node 脚本或 `npm run build` 打包验证）进行 100% 格式安全审查，防止缺失逗号、括号失配等初级语法报错。
3. **保持文本学术流畅度**：所有扩充内容需极具文采与厚重感，体现出严谨的书卷气，全面重塑页面右侧“大事记”的文本质量，提升整款应用的底蕴与品质。
